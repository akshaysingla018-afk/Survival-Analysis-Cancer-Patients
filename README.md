# Survival Analysis of Cancer Patients

## Overview
This project performs survival analysis on cancer patient data to estimate survival probabilities and identify key factors affecting survival.

## Methods Used
- Kaplan-Meier Estimator
- Life Table Method
- Cox Proportional Hazards Model

## Key Findings
- Median survival time: ~1330 days
- Cancer stage significantly affects survival outcomes
- Early-stage diagnosis improves survival probability

## Tools Used
- R
- SPSS

## Features
- Survival curve estimation
- Hazard ratio analysis
- Statistical interpretation of results
